# Image Setup Instructions

## Kitenge Fabrics Display Image

Please add your Kitenge fabrics display image to this `public` folder with the filename:

**`kitenge-fabrics-display.jpg`**

### Image Details:
- **Location:** `kitenge-frontend/public/kitenge-fabrics-display.jpg`
- **Used in:**
  - Homepage "Our Story" section
  - About page "Our Story" section

### Image Description:
The image should show a vibrant display of colorful Kitenge fabrics arranged on shelves, showcasing various African patterns and designs.

### Alternative Formats:
If your image is in a different format (PNG, WebP, etc.), you can:
1. Rename it to `kitenge-fabrics-display.jpg` (or the appropriate extension)
2. Update the image paths in:
   - `src/pages/Home.jsx` (line ~243)
   - `src/pages/About.jsx` (line ~67)

### Recommended Image Specs:
- **Format:** JPG or WebP
- **Size:** 1200x800px or larger
- **Optimization:** Compress for web (keep file size under 500KB for best performance)
